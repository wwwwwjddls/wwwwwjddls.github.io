~ 수익률 계산

입찰가 계산(만원) : 1억-10,000 / 1천-1,000

최저가 취득세율 % 대출률 % 법무비 명도비 관리비 수리비
이자율 % 이자개월 중개율 % 중도상환 % 기타비용 매도가 보증금


* 사용금액 : 취득세 + 법무비 + 관리비 + 명도비 + 수리비 + 이자 + 중도 + 중개 + 기타비용

* 세전수입률 : 세전수입 / ( 입찰금액 - 대출 ) + 사용금액 * 100

* 세후수입률 : 세전수입 / ( 입찰금액 - 대출 ) + 사용금액 - ( 양도세 + 시방세 ) * 100
